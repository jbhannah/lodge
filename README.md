# Lodge

[![Cargo](https://github.com/jbhannah/lodge/workflows/Cargo/badge.svg)][actions-cargo]

Automatable symbolic link manager, à la [GNU Stow][] and [rcm][].

## Installation

    cargo install --git https://github.com/jbhannah/lodge.git

## Usage

    lodge --help
